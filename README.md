# AI 产品经理个人网站

这是一个可直接打开的静态个人网站，面向 AI 产品经理求职或作品集展示。

## 使用方式

直接用浏览器打开 `index.html` 即可预览。

## 需要替换的信息

- `index.html` 里的姓名、邮箱、简历链接、作品集链接
- 三个项目案例中的真实项目名称、职责和产出
- 首页标签或个人定位中的具体经历

## 文件说明

- `index.html`：页面结构与中文文案
- `styles.css`：视觉风格、响应式布局、明暗模式
- `script.js`：移动导航、项目筛选、复制邮箱、滚动动效
- `assets/ai-product-bento.jpg`：深色 Bento 首屏视觉资产
